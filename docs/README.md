# Documentation Hub

This directory contains the active technical documentation for SPACL.

## Start here

- `docs/QUICK_START.md` - first commands to build, test, and run
- `docs/PROJECT_STRUCTURE.md` - source-level architecture (`src/*`)
- `docs/SPACL_ALGORITHM.md` - SPACL algorithm overview

## Scope Of This Public Bundle

This repository intentionally ships a lightweight documentation set.
Internal planning docs, paper-submission docs, and historical benchmark report archives are not included in this public package.

## Benchmark Scripts

Benchmark runner scripts are included under:

- `benchmarks/competitors/scripts/`

## Updating docs

When code or benchmark behavior changes:

1. update `docs/QUICK_START.md` if command usage changed
2. update `docs/PROJECT_STRUCTURE.md` when `src/` structure changes
3. update `docs/SPACL_ALGORITHM.md` when algorithm/API descriptions change
